# x86 Soundscape

An Internet radio and collage.

https://jennymaeleidig.github.io/x86-soundscape/

## Development

Setup:

`npm install --legacy-peer-deps`

Run dev server locally:

`ng serve --open`

Format:

`npm run prettier`

Deploy:

`ng deploy --base-href=/x86-soundscape/`
